We created the medium fidelity prototype in inVision

Prototype

Limitations to our prototype:

This prototype only demonstrates three tasks that represent the major use scenarios.

Tasks

  • Task 1: Send a message(easy) - Sending a message is our most simple and frequently used feature. The user accesses the Chat screen from the Home screen by tapping the speech bubbles in the upper left corner and then selecting the group to chat with. After the selection, the user is presented with the standard messaging screen found on the iPhone. We have not changed this task.

  • Task 2: Send a calendar invite (medium) - The second task is more complex because it involves creating an event, notification and sharing option. The user accesses the Events screen from the Home screen by tapping the calendar icon below the search bar. From the Events screen, the user may review current events or create a new event. We have not changed this task.

  • Task 3: Share a recipe (difficult) - To share a recipe, the user need to create a note first by tapping the note icon below the search bar. Similar to Events, the user may review current notes or create a new note. Lack of a dedicated recipe screen makes the creation process more difficult. We have not changed this task either.
